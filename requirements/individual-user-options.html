2.7. Individual User Options

The scenarios in this section show each individual user's options when using the testing tool.

  • Teacher
    • creates, edits and deletes questions
    • creates, edits and deletes tests
    • takes tests
    • checks grades / test statistics
  • Student
    • takes tests
    • checks his/her grades
  • Grader
    • grades tests
  • System Administrator
    • creates, edits and deletes questions
    • creates, edits and deletes tests
    • takes tests
    • checks grades / test statistics
    • adds a new user
    • edit user's profile
    • releting a user
    • searching for a user
