2.6.1.4 Finding a User

The user database is a bank of users that contains all users that are registered on the Wrecking Ball Testing Tool site. Teachers, Graders, and administrators have access to this User database so that they may look other users. By using this feature, a Teacher may find students to add to the course, or an administrator may find a User to edit the preferences.
