1.5.4. Quiz Work

Quiz Work is, one of many, online test-making tools that allows users to both create and search for user generated tests. Users may take the tests in a variety of different formats, including games. It is in the users' power to create the questions and form a question bank from which random tests may be generated, completed, and scored. Quiz Work also offers a several different premium services for teacher that allows them to track the progress of their students and gather data as to how the class is performing throughout the course.

Good Features:

  • different types of questions (e.g. multiple choice, true/false, short answer, long answer, etc.);
  • URL of a particular test can be shared with other users;
  • this testing tool provides the options to practice for an exams in the form of learning games;
  • provides the user with a score after finishing the tests;
  • test results can be ranked/compared after the test was completed;
  • allows users to create their own profiles and save any tests;
  • allows teachers to create classes so that their students can easily have access to tests;
  • users may take the test as many times as they like;
  • supports image questions;
  • option to change editability and privacy of the tests;
  • supports multiple languages;
  • the test tool has a timer;
  • test is avalable for a limited amount of time (i.e. from September 25, 20XX to September 27, 20XX).
Bad Features:
  • does not allow you to save the progress of a test (i.e. a student may not be able to continue where they left off if they leave theh page);
  • each question has a time limit, therefore a test taker cannot skip a question and come back to it later.
Missing Features:
  • no public question bank;
  • tests cannot be described by difficulty.
