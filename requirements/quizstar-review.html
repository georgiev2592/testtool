1.5.6. QuizStar

QuizStar is a free online test-making tool that allows users to create accounts and easily make tests in several languages. Students can take tests easily and receive feedback, $

Good Features:

  • allows user to create different types of tests/
  • easy for teachers to manage classes and quizzes.
  • you may compare your grades to other students in your class.
  • easy to specify the amount of attempts a student can try.
  • provides the user with immediate feedback.
  • allows teachers to create their own profiles and save any tests.
  • supports questions in several different languages.
  • option to change editability and privacy of the tests.
  • teachers can specify the number of questions displayed per page.
  • can receive emails about grades.
  • can start and pause tests.
  • can have images in the tests.
Bad Features:
  • no database to share tests between teachers.
Missing Features:
  • does not provide a test timer.
  • no feature to determine the difficulty of the tests.
  • database of all registered teachers tests.
