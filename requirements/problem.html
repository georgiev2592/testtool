1.1. Problem Statement

The general problems to be addressed by the Testing Tool project are as follows:

  1. to ease teachers in the process of test making as well as help in the time-consuming grading of these tests
  2. to provide a database of all test questions used in our tool to aid all teachers using our tool in creating the most diverse and comprehensive tests

While these problems have been addressed by several other available products, the Testing Tool provides a solution with major improvements over existing tools. Specific areas of focus are the following:

  1. creating a new, efficient way to test computer science majors
  2. compiling all test questions for anyone to use
  3. easily generate tests from existing teachers teaching the same class

The Testing Tool project addresses the requirements of a particular customer. This customer needs a tool with a certain specification not available in any other tool. This issue is now been addressed and the challenge will be to combine existing features with that of the new features not yet implemented in any other product.
