High-level Architectural Design of the Wrecking Ball Test Tool

This design documentation describes the packages of the Test Tool software system. The system is comprised of three application programs:

  • the Test Tool program, for Teachers to create and manage a question bank and test bank
  • the Test Tool Administration program, for administrators to manage databases and control other functions of Test Tool central hosts
  • the Calendar Tool Server, which runs in the background on central hosts to control communication among users' local computers and the host
See the companion requirements specification document has complete details of calendar tool functionality.
