1.5.1. Class Marker

Class Marker is an online test making tool with a question bank that allows users to create questions and tests as well as distribute them. Users can create tests with serveral normal question types and assign the test to a group of users or give out the links. This tool also as an option to make the test takers pay to take the test as well as get a certificate after successfully completing the test.

Good Features:

  • decent amount of questions supported (multiple choice, true/false, grammar, free text, essay)
  • can share test by creating accounts for test takers or by sharing a link
  • has basic question bank functionality with custom categories and searching
  • provides the user with a score after finishing the tests
  • many options for how the test is taken (timed, number of tries)
  • supports images in all questions
  • options for giving automatic feedback when test taker answers specific question wrong or right
  • can export and import questions for text files
Bad Features:
  • Grammar and Free Text questions depend on exact string matching with the user's answer which can be fragile
  • does not offer very good previewing of questions (the questions don't look the same as the student sees)
Missing Features:
  • no sharing of questions between teachers
  • doesn't have very many question types
