2. Functional Requirements

The primary functionality the TestTool system is for creating test and quizes. The TestTool operations are performed by individual users on their personal accounts, as well as by administrative users, who may create tests for groups of users. The TestTool system also provides functionality for system administration and creating a variety of test options.

Following an overview of the TestTool user interface, details of system use are presented in the following scenarios:

  • an individual user scheduling an appointment
  • an individual user browsing tests in a variety of ways
  • a system administrator updating the question databases
  • an individual user setting system options
  • details of file and edit commands
These scenarios cover all functional requirements for the system.
