1.5.5. Quizlet

Quizlet is a widely populat online test-making tool that allows users to create profiles and both create and search for user generated tests. Users may take the tests in a variety of different formats, including games. It is in the users' power to create the questions and form a question bank from which random tests may be generated, completed, and scored. Quizlet also offers a premium service for teacher that allows them to track the progress of their students and gather data as to how the class is performing on the questions. Ultimately, Quizlet is an exceptional testing tool that allows users to effectively practice and take tests in a variety of different subjects.

Good Features:

  • allows user to create many different formats of tests (e.g. fill-in-the-blank, flashcards, etc.).
  • users may share the tests they generate with other users.
  • takes away the monotony of studying through generated games that test knowledge.
  • you may compare your grade to other users have taken the test.
  • allows the user to limit the amount of questions given at a time (i.e. only being provided a random, select amount of questions from a question bank).
  • very simple and easy to use.
  • provides the user with a score after finishing the tests.
  • allows users to create their own profiles and save any tests.
  • allows teachers to create classes so that their students can easily have access to tests.
  • users may take the test as many times as they like.
  • supports image and audio questions.
  • option to change editability and privacy of the tests.
  • supports various languages.
  • appealing interface.
Bad Features:
  • since there are many users and public tests, users easily run into questionable or wrong information.
  • does not allow you to save the progress of a test (i.e. a student may not be able to continue where they left off if they leave theh page).
  • users see the answers to the test before they actually choose a test format which may lead to an inaccurate score.
Missing Features:
  • does not provide a timer, which is unrealistic towards a real testing environment.
  • no way to set a due date for tests.
  • lacks a feature to determine the difficulty of the tests.
