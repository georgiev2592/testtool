2.2.1.1. Multiple Choice

To add questions, the user chooses the Add Questions option from the menu page under the Question Manager popup menu.

Figure 9: Initial State of the Create Question Form


Select the correct question type by clicking on the "Question Type" drop down menu. After the type has been selected by clicking on it, the necessary fields for entering the question parameters will appear.


Figure 10: Multiple Choice Option Selected


Enter the question prompt into the labeled text area.


Figure 11: Question Prompt Entered


Add answers to the table below by entering the text into the "answer" column and selecting the radio button in the "correct?" column if the new answer is correct.


Figure 12: Question Prompt Entered


Answers can be added or removed with the "+" and "-" symbols respectively in the leftmost column.


When finished, click "save" to save the question and exit the form.
