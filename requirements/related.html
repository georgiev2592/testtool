1.5. Related Systems

Widely used commercial systems that provide functionality comparable to the Test Tool include:

  • Class Marker
  • Testmoz
  • Easy Test Maker
  • Quiz Work
  • Quizlet
  • Quiz Star

Each of these systems provides a wide variety of testing functionality. In terms of such functionality, the goal for the Testing Tool is to provide the important core features found in these tools, not the full range of features found in all of them. The Testing Tool does provide modest functional improvements in three areas, when compared to most commercially available tools. These improvements are:

  1. share questions between users;

  2. export questions;

  3. support for programming questions.

A detailed review of each related tool follows in these subsections:

  • Class Marker
  • Testmoz
  • Easy Test Maker
  • Quiz Work
  • Quizlet
  • Quiz Star
Following these reviews, a summary of the features is provided in a feature comparison matrix.
