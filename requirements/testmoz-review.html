1.5.2. Testmoz

Testmoz is a simple online tool for creating and distributing digital tests. Most of the features of the tool are free, but more for more comprehensive test management features (such as a way to display all authored tests in one place) users must pay a $20.00/year fee. Testmoz provides an acceptable level of testing tool functionality, but it missing a few key features, most notably the question bank of our TestTool design.

Good Features:

  • Provides simple and easy to use tools for creating digital tests.
  • Supports a number of different question types, including true/false, multiple choice, and fill-in-the-blank.
  • Questions can be individually weighted (assigned point values).
  • Each test is tied to a unique url for simple distribution.
  • Tests are password protected so that only students with the url and corresponding password can access them.
  • Accounts are password protected so that tests can only be editted by the users that create them.
  • Test grading is automatic and scores are available to students immediately after submitting their answers.
  • Tests can be editted even after they have been created/published.
  • Provides public interface for viewing all tests that a user has created together.
Bad Features:
  • Questions are specfic to each test, meaning that they have to be re-created for each test that re-uses them.
  • Students can only view their scores immediately after taking a test.
Missing Features:
  • No built-in timer for test-taking.
  • Missing support for a number of common question types, including matching and short-answer.
  • No way for teachers to award partial credit or add handwritten notes to students' test results.
  • No built-in option to add images to questions.
